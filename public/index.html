Najdi volný název i doménu

Zadej zaměření → vygeneruju značkové názvy a u každého zkontroluju volné domény a stopu na webu. Ideál = volná .com/.cz + minimum výskytů na webu (jako „faxxar“).

Jak to funguje
  • Názvy generuje Cloudflare Workers AI (Llama) podle tvého zaměření.
  • Dostupnost domén se zjišťuje přes RDAP (404 = volná, 200 = zabraná). Autoritativně pro .com/.cz/.net.
  • Stopa na webu = kolik z top výsledků Brave Search obsahuje to jméno. Méně = čistší značka.
Jak se počítá skóre
  • 55 % = dostupnost domén (.com a .cz mají největší váhu).
  • 45 % = nízká stopa na webu (0 výskytů = plný bod).
  • ★ = nejlepší kandidát v seznamu.
Web stopa nefunguje?
  • Bez klíče appka běží dál — jen bez sloupce Web (skóre je pak jen podle domén).
  • Zapnutí: Brave Search API klíč jako secret BRAVE_API_KEY. Postup v docs/BUILD.md.
Bezplatné?
  • Cloudflare Worker + Workers AI = free tier. RDAP = zdarma. Brave Search = ~1000 dotazů/měsíc v rámci free kreditu.
